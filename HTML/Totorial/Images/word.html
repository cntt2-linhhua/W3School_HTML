1. img : Sử dụng phần tử để xác định hình ảnh 2. src : Sử dụng thuộc tính src HTML để xác định URL của hình ảnh 3. alt : Sử dụng thuộc tính alt HTML để xác định văn bản thay thế cho hình ảnh, nếu nó không thể được hiển thị 4. width và height : để xác định kích thước của hình ảnh 5. float : Sử dụng thuộc tính float CSS để cho hình ảnh nổi
